# fakebook
 fakebook: a web application for a community to communicate and share ideas like school group.
 #Features:
 *Login
 *Signup
 *User authentication
 *User profile editing
 *Messaging
 *Posting 
 *News Feeds
 #How to use:
 1. Clone the repository
 2. open the file in a code editor like vs code
 3. navigate to the backend folder and type the following command: npm start
 4. after the server started, navigate to the frontend folder seperate terminal or the vs code intergrated terminal and type the following command: npm run dev
 5. click the link given by the terminal, use valid credentials to signup, or use credentials in the user.json file in the backend
